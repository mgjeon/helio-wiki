# Heliosphere Model

The source list frames a heliosphere modeling chain as magnetic maps → coronal domain → model interface → heliospheric domain. Source: raw/transcripts/Solar-Physics-Codes-Solar-Wind-CME_1.md

- Magnetic-map inputs named in the source list are GONG, MDI, and HMI. Source: raw/transcripts/Solar-Physics-Codes-Solar-Wind-CME_1.md
- Coronal-domain models named in the source list are PFSS and MAS. Source: raw/transcripts/Solar-Physics-Codes-Solar-Wind-CME_1.md
- WSA is listed as the model interface between coronal and heliospheric domains. Source: raw/transcripts/Solar-Physics-Codes-Solar-Wind-CME_1.md
- ENLIL and HUX are listed as heliospheric-domain models. Source: raw/transcripts/Solar-Physics-Codes-Solar-Wind-CME_1.md
